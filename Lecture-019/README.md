# C++ Containers

This document outlines the hierarchy of containers in C++.

## Containers
- **Sequence Containers**
  - Array
  - Vector
  - Deque
  - List
  - forward_list

- **Container Adaptors**
  - Stack
  - Queue
  - Priority Queue

- **Associative Containers**
  - Set
  - Map
  - MultiSet
  - MultiMap

- **Unordered Associative Containers**
  - Unordered Set
  - Unordered Map
  - Unordered MultiSet
  - Unordered MultiMap
